# CatalogMovie

Aplikasi ini merupakan apliaski Android Native yang menampilkan list Movie dan TV Show dari API https://www.themoviedb.org/. Aplikasi dikembangkan dengan menggunakan:

<ul>
  <li>
    Programming Language
    <ul>
      <li>
        Kotlin
      </li>
    </ul>
   </li>
   <li>
    IDE
    <ul>
      <li>
        Android Studio
      </li>
    </ul>
   </li>
   <li>
      Design Pattern 
    <ul>
      <li>
        MVVM
      </li>
    </ul>
   </li>
   <li>
      Design Pattern 
    <ul>
      <li>
        MVVM
      </li>
    </ul>
   </li>
   <li>
      Library
    <ul>
      <li>
        Retrofit
      </li>
      <li>
        Koin
      </li>
      <li>
        Binding
      </li>
    </ul>
   </li>
</ul>
